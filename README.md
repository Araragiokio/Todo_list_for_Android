# TodoApp 🗡️

A beautiful anime-inspired productivity app built with React Native + Expo.

## Designed & Developed by
**Aditya Gupta**

## Features
- ✅ Task management with categories & priorities
- 🔔 Smart reminders & alarms
- 🌸 Anime Lottie animations
- 🏅 Badge & achievement system
- 📊 Progress tracking & yearly heatmap
- 🌙 Light & dark mode
- ☁️ Cloud sync via Firebase

## Built With
- React Native
- Expo
- Firebase
- Lottie
- React Navigation

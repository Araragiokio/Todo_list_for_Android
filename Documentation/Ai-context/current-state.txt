## Project Setup

## Navigation System

The navigation structure of the application is fully implemented.

Navigation uses:

Expo Router
React Navigation

The application uses **bottom tab navigation**.

Main screens currently available:

Home
Add Task
Reminders
Settings
Profile

Navigation layout files are located in:

app/_layout.tsx
app/(tabs)/_layout.tsx

---

## Theme System

A global theme system has been implemented.

Supported modes:

Light theme
Dark theme
System theme

Theme preferences are stored using AsyncStorage so that the selected theme persists between sessions.

Theme colors are defined in:

constants/Colors.ts

---

## Splash Screen

The application includes a cinematic splash screen sequence.

Animations used in the splash screen:

document-checklist.json
working-desk.json
confetti.json

The splash screen appears when the app launches and transitions smoothly into the Home screen.

## Task Data Model

The full task structure has already been designed and implemented.

The task model supports:

Task title
Category
Tags
Energy level
Priority level
Due date
Reminder
Notes
Subtasks
Recurring rules
Completion state
Manual sorting order

The data model is defined in:

types/Task.ts

---

## Task Storage System

Tasks are currently stored locally using AsyncStorage.

The storage system supports the following operations:

getTasks()
addTask()
toggleTask()
deleteTask()
editTask()
toggleSubtask()
updateSortOrder()

The storage system also supports:

custom categories
task search
completed task counting

All storage logic is implemented in:

storage/TaskStorage.ts

---

# What TodoApp Is

TodoApp is a **mobile productivity application** designed to help users organize tasks, track progress, and maintain daily productivity.

Unlike traditional todo list applications, TodoApp focuses on making productivity feel **engaging and emotionally rewarding** rather than mechanical.

The application combines:

Task management
Light gamification
Visual feedback
Clean mobile design

to create a productivity tool that feels like a **daily companion**.

---

# Core Idea Behind the App

Most productivity tools focus only on efficiency.

TodoApp also focuses on **emotional motivation**.

The app aims to make users feel:

Accomplished when tasks are completed
Encouraged to maintain streaks
Rewarded through visual feedback

Small design elements such as animations, progress tracking, and achievements are intended to make the experience feel more engaging.

---

# Target Users

TodoApp is designed primarily for individuals who want a simple but powerful productivity system.

Example users include:

Students managing assignments and study tasks
Developers tracking projects and coding work
Professionals organizing daily responsibilities
Individuals tracking personal goals and habits

The goal is to support productivity across different life areas.

---

# Core Functionality

The core functionality of TodoApp revolves around **task management**.

Each task can include:

Title
Category
Tags
Priority level
Energy level
Due date
Reminder
Notes
Subtasks
Recurring rules

These attributes allow tasks to be organized and filtered in multiple ways.

---

# Organizational Systems

The app includes several systems to help users organize their tasks.

Categories
Used for broad grouping of tasks.

Tags
Allow flexible labeling and search.

Energy Levels
Help users choose tasks based on available energy.

Sorting
Allows tasks to be arranged by priority, deadline, or custom order.

---

# Design Philosophy

TodoApp follows several design principles.

Simplicity
The interface should remain clean and easy to understand.

Motivation
Visual feedback should encourage task completion.

Clarity
Users should immediately understand their current workload.

Minimal Distraction
Animations and visuals should enhance the experience without overwhelming the interface.

# Technology Stack

The application is built using modern mobile development tools.

Framework:

React Native + Expo

Language:

TypeScript

Local Storage:

AsyncStorage

Navigation:

Expo Router / React Navigation

Animations:

Lottie

These tools allow the app to be developed quickly while maintaining good performance and maintainability.

# TodoApp — Animation System

## Overview

Animations in TodoApp are used to enhance the user experience by providing visual feedback and making interactions feel more satisfying.

The application uses **Lottie animations** to display high-quality, lightweight animations that run smoothly on mobile devices.

Animations are used sparingly and only for meaningful events to avoid distracting the user.

---

# Animation Technology

Animations are implemented using the following library:

lottie-react-native

Lottie allows animations exported from tools like **Adobe After Effects** to be played directly in React Native applications using JSON files.

Advantages of Lottie:

* Lightweight animation files
* Smooth performance
* Easy integration with React Native
* Scalable across different screen sizes

---

# Animation Storage

All animation files are stored in the following directory:

assets/animations/

Example structure:

```
assets/
└── animations/
    ├── cherry-blossom.json
    ├── confetti.json
    ├── document-checklist.json
    └── working-desk.json
```

Each JSON file contains animation instructions used by the Lottie player.

## Sakura Background Animation

Animation file:

cherry-blossom.json

Purpose:

Provides a subtle animated background on the Home screen.

Behavior:

* Sakura petals slowly fall across the screen
* Animation runs continuously
* Opacity is reduced to avoid distracting the user

Recommended opacity:

0.12

This creates a calm visual atmosphere while keeping the UI readable.

# Animation Design Principles

Animations in TodoApp follow several design principles.

Purposeful
Animations should occur only when they communicate meaningful events.

Subtle
Background animations should never distract from the main UI.

Performance-friendly
Animations should remain lightweight to ensure smooth performance on mobile devices.

Rewarding
Completion and achievement animations should feel satisfying and motivating.
